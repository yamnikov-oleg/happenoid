# Happenoid
Маленький клон ithappens.me

Для запуска на своей машине:
1. Установить Ruby 4.1.6
2. Установить Rails 4.2.3
3. Клонировать репозиторий в локальную папку
4. ```cd happenoid```
5. ```bundle install```
6. По умолчанию проект настроен на работу с sqlite
7. При необходимости интеграции с mysql заменить ```db/database.yml``` на ```db/mysql_database.yml``` и убедиться, что проект будет иметь доступ к бд с указанными настройками
8. ```rake db:setup``` для создания баз и таблиц
